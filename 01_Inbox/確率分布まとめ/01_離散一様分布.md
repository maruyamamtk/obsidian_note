# 離散一様分布 (Discrete Uniform Distribution)

#統計

## 概要

離散一様分布は、確率変数がとりうる有限個の各値が、すべて等しい確率で発生する場合に用いられる離散型確率分布です。最もシンプルな分布の一つであり、他の概念の基礎として重要な役割を果たします。

## 基本情報

- **種別**: 離散型
- **定義域**: $\{k_1, k_2, ..., k_N\}$（有限個の離散値）
- **パラメータ**: N（取りうる値の総数）

## 数学的性質

### 確率質量関数 (PMF)
$$P(X = k) = \frac{1}{N}$$

ここで、Nは確率変数Xの取りうる値の総数です。

### 統計的性質

- **平均（期待値）**: $E[X] = \frac{k_1 + k_2 + ... + k_N}{N}$
- **分散**: $\text{Var}[X] = \frac{N^2 - 1}{12}$（連続する整数 1, 2, ..., N の場合）
- **標準偏差**: $\sqrt{\frac{N^2 - 1}{12}}$

### 特殊ケースの例
6面体のサイコロの場合（N=6, 値は1,2,3,4,5,6）：
- **平均**: $(1+2+3+4+5+6)/6 = 3.5$
- **分散**: $(6^2-1)/12 = 35/12 ≈ 2.917$

## 特徴と性質

### 基本的特徴
- **確率密度が一定**: すべての値が等しい確率 $1/N$ を持つ
- **完全な不確実性**: 事前情報が全くない状態を表現
- **対称性**: 中央値を中心に完全に対称

### 数学的特性
- **再生性**: 一般に持ちません
- **無記憶性**: 持ちません
- **最大エントロピー**: 有限離散値の中で最大のエントロピーを持つ

## モデリングへの応用

### 適用例

#### ゲーム・確率
- **公正なサイコロ**: 各面が出る確率が等しい（1/6ずつ）
- **公正なコイン**: 表と裏が等確率（1/2ずつ）
- **ルーレット**: 等分された区間への停止確率
- **カードゲーム**: シャッフルされたカードの順序

#### 計算機科学・情報技術
- **乱数生成**: コンピュータの疑似乱数生成の基礎
- **ハッシュ関数**: データの均等な分散
- **暗号学**: 暗号鍵の生成における基礎分布

#### 統計・確率論の基礎
- **標本抽出**: 無作為抽出の理論的基礎
- **確率の基本概念**: 等確率事象の表現
- **モンテカルロ法**: シミュレーションの基礎分布

### 実世界での例

1. **宝くじ**: 番号が機械的に選ばれる場合
2. **調査**: 無作為に調査対象を選択
3. **実験**: ランダムな条件の割り当て
4. **ゲーム**: 公平な選択やランダムな順序決定

## 理論的重要性

### 事前分布としての役割
- **無情報事前分布**: 事前知識がない場合の初期設定
- **ベイズ統計**: 主観的確率の出発点
- **最大エントロピー原理**: 制約下で最も「無偏」な分布

### 他の分布との関係基礎
- **連続一様分布**: 連続版の離散一様分布
- **多項分布**: すべてのカテゴリが等確率の場合
- **正規分布**: 中心極限定理により収束先となる場合がある

## 数学的展開

### 一般化された形
区間 $[a, b]$ の整数値を取る場合：
- **定義域**: $\{a, a+1, a+2, ..., b\}$
- **総数**: $N = b - a + 1$
- **平均**: $E[X] = \frac{a + b}{2}$
- **分散**: $\text{Var}[X] = \frac{(b-a+1)^2 - 1}{12}$

## 計算例

### 例題1: サイコロ
6面サイコロで偶数が出る確率は？
- 偶数: {2, 4, 6}（3個）
- 全事象: {1, 2, 3, 4, 5, 6}（6個）
- 確率: $P(\text{偶数}) = \frac{3}{6} = \frac{1}{2}$

### 例題2: 期待値の計算
1から10までの整数から1つを選ぶ場合の期待値：
$$E[X] = \frac{1+2+...+10}{10} = \frac{55}{10} = 5.5$$

## 限界と適用条件

### 適用が適切な条件
- すべての結果が本当に等確率である
- 事前情報が全くない
- 有限個の離散値を扱う

### 適用に注意が必要な場合
- 実際には確率に偏りがある場合
- 継続的な学習や経験が影響する場合
- 物理的制約により等確率でない場合

## 実装例（概念的）

```python
import random

def discrete_uniform_pmf(k, values):
    """離散一様分布の確率質量関数"""
    if k in values:
        return 1.0 / len(values)
    else:
        return 0.0

def discrete_uniform_properties(values):
    """離散一様分布の統計的性質"""
    n = len(values)
    mean = sum(values) / n
    
    # 分散の計算
    variance = sum((x - mean)**2 for x in values) / n
    std = variance ** 0.5
    
    return {
        'mean': mean,
        'variance': variance,
        'std': std,
        'n_values': n
    }

def generate_discrete_uniform(values, size=1):
    """離散一様分布からの乱数生成"""
    return [random.choice(values) for _ in range(size)]

# 使用例
die_values = [1, 2, 3, 4, 5, 6]
properties = discrete_uniform_properties(die_values)
print(f"サイコロの期待値: {properties['mean']}")  # 3.5
```

## 中心極限定理との関係

離散一様分布であっても、標本サイズが大きくなると標本平均は正規分布に近づきます：

$$\bar{X} \xrightarrow{d} N\left(\mu, \frac{\sigma^2}{n}\right)$$

ここで、$\mu$ と $\sigma^2$ は離散一様分布の平均と分散です。
